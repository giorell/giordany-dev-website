Contract Manager App

The Contract Manager app is a faster way to fill out short-term leases for rental units, intended for landlords in the Netherlands. Built for a client in Amsterdam, it has the capability to keep track of addresses, tenants, and their respective contracts. It can generate a PDF that can be sent to both the landlord and the tenant, and a secondary PDF for tenant registration with the city.

Technologies

  • Coordinator-Pattern
  • Core Data
  • Signature capability
  • Camera capability

Fantasy Football Demo App

Built as a proof-of-concept, this fantasy football tracker is designed to allow up to 6 users to play as a management team for an American Football team during Football season. With 24 positions available, users are only allowed to fill 4 positions with athletes from the football draft. This app is made posible by utilizing Google Firebase.

Technologies

  • Google Firebase
  • Authorized login
  • updates in real-time

Ultimate Movie Database

Built for the Udacity Captstone project, this app utilizes themoviedb.org API. The goal of the app for a user is to view the latest trending movies, to add movies to a watchlist or a favorite list, and to view their profile.

Technologies

  • RESTful API calls
  • Core Data
  • Collection Views
  • Table Views
